This folder contains 5 sub-folders.


"commutingTime" contains a Python script for getting commuting times for each property with Google Map Distance Matrix API;

"K-means&crime" contains Python script for crime score calculation and K-means clustering;

"propertyScrape" contains Python scripts for scraping property information from Zillow.com;

"yelp" contains Python scripts for getting nearby retailer information for each property with Yelp API;

"data" contains .tsv or .json files, which are the data we got.
